Vulkan engine is a modified version of a graphics engine by Niko Procopi, which can be found here:
https://github.com/IGME-RIT/VkHitboxes.

The final scene has proper three point lighting, moving camera, spinning fidget spinner, and basic color texturing. To change around the scene, adjust the ray.frag shader, and run the _compileRayShaders.cmd command file in order to compile.
